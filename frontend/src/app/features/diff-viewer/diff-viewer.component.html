How to read this page
  1. Pick a demo above (start with GW150914), or search the catalog / Advanced GPS.
  2. Press Run analysis (cold GWOSC fetch can take a few minutes).
  3. Read Production vs Independent in coincidence — then hover charts. Dashed line = event time (t=0).
Three panes
Raw strain S_raw, predicted noise N̂, residual R = S_raw − N̂.
Blind search
Template-free excess power on the residual, compared to a noise-calibrated FAR threshold. A residual-only trigger means subtraction revealed energy that raw strain missed.
Production
Dual-detector rule: coherent residual EP + lag within light-travel window + envelope timing veto.
Independent
Both IFOs clear the single-detector residual gate on their own. Diagnostic only — not required for Production (e.g. GW150914).
Raw coincidence
Both detectors trigger on unsubtracted strain excess power.
